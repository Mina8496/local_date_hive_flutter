# Local Data Storage with Hive Flutter

A Flutter project demonstrating local data persistence using Hive and Hive Flutter.

## Features

* Store and retrieve data locally using Hive.
* Perform CRUD operations (Create, Read, Update, Delete).
* Fast and lightweight local database solution.
* Clean and organized project structure.
* Beginner-friendly implementation for learning local storage.

## Tech Stack

* Flutter
* Dart
* Hive
* Hive Flutter

## Getting Started

### Prerequisites

* Flutter SDK installed
* Dart SDK
* Android Studio or VS Code

### Installation

1. Clone the repository:

```bash
git clone https://github.com/Mina8496/local_date_hive_flutter.git
```

2. Navigate to the project directory:

```bash
cd local_data_hive_flutter
```

3. Install dependencies:

```bash
flutter pub get
```

4. Run the application:

```bash
flutter run
```

This project was built as part of a Flutter learning journey to:
